## 1. MUST-DO чеклист (что ОБЯЗАТЕЛЬНО внедрить)

### Онбординг

- [ ] **Quiz-based онбординг 15–25 экранов** — health/wellness приложения (Noom 80+, Flo 70+) показывают, что длинный персонализированный quiz повышает perceived value и conversion ([Retention Blog](https://www.retention.blog/p/the-longest-onboarding-ever))
- [ ] **Прогресс-бар с указанием времени** — снижает drop-off до 40% ([Retention Blog — Headway](https://www.retention.blog/p/headway-evolution-2024-2025))
- [ ] **Чередование вопросов и мотивационных экранов** — баланс ask/give, как Headway ([Retention Blog — Headway](https://www.retention.blog/p/headway-evolution-2024-2025))
- [ ] **Loading screen "Создаём ваш план..."** после quiz — +10–20% конверсия (Noom pattern, [Retention Blog](https://www.retention.blog/p/the-longest-onboarding-ever))
- [ ] **Персонализированный результат ДО paywall** — профиль тяги, прогноз, план → invested buy-in ([Lenny's Newsletter](https://www.lennysnewsletter.com/p/mastering-onboarding-lauryn-isford))
- [ ] **Запрос push permissions В КОНТЕКСТЕ** (после SOS, не в начале) — ранний запрос = отказ навсегда ([Appcues](https://www.appcues.com/blog/mobile-permission-priming))

### Paywall

- [ ] **Hard paywall после онбординга с 7-дневным free trial** — hard paywall конвертирует 5–6x лучше soft; 82–89% триалов начинаются в Day 0 ([DEV Community](https://dev.to/paywallpro/the-paywall-timing-paradox-why-showing-your-price-upfront-can-5x-your-conversions-4alc), [RevenueCat 2025](https://www.revenuecat.com/state-of-subscription-apps-2025/))
- [ ] **2 тира: месячный ($9.99) + годовой ($79.99)** — показ двух планов рядом даёт +31% к годовым подпискам ([Airbridge](https://www.airbridge.io/blog/mobile-paywall-optimization-actionable-tips))
- [ ] **Анимированный paywall** — 2.9x конверсия vs статический ([Adapty](https://adapty.io/blog/how-to-design-a-paywall-for-a-mobile-app/))
- [ ] **Персонализация на paywall** (имя, тип тяги) — +17% конверсия ([Adapty](https://adapty.io/blog/how-to-design-a-paywall-for-a-mobile-app/))
- [ ] **Social proof с конкретными числами** — +72% install-to-trial ([Adapty 2026](https://adapty.io/state-of-in-app-subscriptions/))
- [ ] **Benefit-driven CTA** — "Начать мой план бесплатно" вместо "Подписаться" ([DEV Community](https://dev.to/paywallpro/subscription-onboarding-15-patterns-you-must-know-4n4f))
- [ ] **"Honest Paywall"** — прозрачный таймлайн trial (Day 1 → Day 5 → Day 7) → +23% конверсия, –55% жалоб (Blinkist pattern, [DEV Community](https://dev.to/paywallpro/subscription-onboarding-15-patterns-you-must-know-4n4f))
- [ ] **Value anchoring** — "$80/год vs $10K+/год лечение диабета" ([Product Vision](../02-product/PRODUCT-VISION.md))

### Retention

- [ ] **Streak system с Streak Freeze** — пользователи с 7+ дней streak 3.6x более вероятно остаются; Freeze –21% churn ([StriveCloud](https://www.strivecloud.io/blog/gamification-examples-boost-user-retention-duolingo), [Orizon](https://www.orizon.co/blog/duolingos-gamification-secrets))
- [ ] **Push-уведомления: макс 1/день, персонализированные, ≤90 символов** — пользователи с push вкл. показывают 3–10x retention ([ContextSDK](https://contextsdk.com/blogposts/the-psychology-of-push-why-60-of-users-engage-more-frequently-with-notified-apps))
- [ ] **Daily check-in + curriculum** (5+ мин сессия) — 35% D30 retention vs 22% для коротких сессий ([Lucid](https://www.lucid.now/blog/retention-metrics-for-fitness-apps-industry-insights/))
- [ ] **Milestone badges** (7, 14, 30, 90 дней) привязанные к реальным достижениям — +30% completion rate ([Orizon](https://www.orizon.co/blog/duolingos-gamification-secrets))

### ASO

- [ ] **Keywords в Title** — самый важный фактор ранжирования; Title: "Sugar Quit: AI Craving Coach" ([ASOMobile](https://asomobile.net/en/blog/aso-in-2026-the-complete-guide-to-app-optimization/))
- [ ] **Рейтинг ≥4.5 звезды** — падение ниже 4.0 = "обрыв" конверсии –15–20% ([Appalize](https://www.appalize.com/da/blog/app-marketing/app-store-ratings-impact-on-downloads-data-driven-analysis))
- [ ] **6–8 скриншотов с benefit-driven текстом** — скриншоты могут увеличить конверсию на 20–35% ([Wezom](https://wezom.com/blog/app-store-optimization-aso-in-2025-a-complete-guide-for-growth))
- [ ] **App Preview видео** — 62% приложений без видео = quick win; конверсия +10–30% ([Business of Apps](https://www.businessofapps.com/marketplace/app-store-optimization/research/app-store-optimization-statistics/))
- [ ] **Ответы на отзывы в течение 24 часов** — 2x эффект vs ответ через неделю; +0.1–0.3 к рейтингу за 6 мес ([Appalize](https://www.appalize.com/da/blog/app-marketing/app-store-ratings-impact-on-downloads-data-driven-analysis))

## 2. AVOID чеклист (чего ИЗБЕГАТЬ)

- [ ] **НЕ требовать регистрацию до демонстрации ценности** — отпугивает до 75% пользователей ([Decode Agency](https://decode.agency/article/app-onboarding-mistakes/))
- [ ] **НЕ запрашивать push permissions на первом экране** — после отказа изменить решение крайне сложно ([Appcues](https://www.appcues.com/blog/mobile-permission-priming))
- [ ] **НЕ описывать фичи, а не ценность** — "AI-коуч помогает победить тягу" вместо "встроенный чат с AI" ([Decode Agency](https://decode.agency/article/app-onboarding-mistakes/))
- [ ] **НЕ отправлять >1 push/день** (кроме craving prediction) — 46% отключат при 2–5/неделю ([Mobiloud](https://www.mobiloud.com/blog/push-notification-statistics))
- [ ] **НЕ добавлять gamification без привязки к реальным действиям** — badges за "открыл настройки" = fluff, бейдж за "7 дней без сахара" = реальное достижение ([DEV Community](https://dev.to/devang1810/how-mobile-app-gamification-improves-user-retention-414a))
- [ ] **НЕ использовать >3 тиров подписки** — decision paralysis (Закон Хика-Хаймана, [Qonversion](https://qonversion.io/blog/how-to-design-paywall-that-converts))
- [ ] **НЕ делать trial <4 дней** — 55% отмен 3-дневного trial происходят в Day 0 ([RevenueCat 2025](https://www.revenuecat.com/state-of-subscription-apps-2025/))
- [ ] **НЕ откладывать paywall** — каждый день задержки = потеря конверсионного окна; 82–89% триалов = Day 0 ([RevenueCat 2025](https://www.revenuecat.com/state-of-subscription-apps-2025/))
- [ ] **НЕ копировать скриншоты 1:1 между App Store и Google Play** — разные правила по текстовой площади, видимости, индексации ([AppTweak](https://www.apptweak.com/en/aso-blog/how-to-optimize-your-app-screenshots))
- [ ] **НЕ считать retention маркетинговой проблемой** — retention = product problem; прогресс, контроль и reward > push spam ([Codeft](https://codeft.io/think-tank/push-notifications-not-pushy-apps-how-to-optimize-mobile-app-development-for-retention/))

## 3. Рекомендации по экранам

| Экран | Что должно быть | Источник/обоснование |
|-------|----------------|---------------------|
| **Welcome** | Приветствие + обещание ("3 минуты → ваш персональный план") + прогресс-бар | Прогресс-бар –40% drop-off ([Retention Blog](https://www.retention.blog/p/headway-evolution-2024-2025)) |
| **Quiz блок 1: Цели** | 3–4 экрана — "Зачем бросаете сахар?" (визуальные карточки) | Персонализация +82% retention ([UserGuiding](https://userguiding.com/blog/user-onboarding-statistics)) |
| **Мотивационный экран 1** | Social proof: "127,000+ пользователей уже сократили сахар" | Social proof +72% install-to-trial ([Adapty 2026](https://adapty.io/state-of-in-app-subscriptions/)) |
| **Quiz блок 2: Привычки** | 3–4 экрана — триггеры, потребление, время тяги | Каждый вопрос deepens commitment ([Flo Health](https://medium.com/flo-health/mobile-onboarding-evolution-part-1-cfc9702835ce)) |
| **Мотивационный экран 2** | "75% американцев хотят сократить сахар. Вы не одиноки" | Чередование ask/give ([Retention Blog — Headway](https://www.retention.blog/p/headway-evolution-2024-2025)) |
| **Quiz блок 3: Контекст** | 3–4 экрана — образ жизни, прошлые попытки, мотивация | Quiz deepens commitment ([Retention Blog](https://www.retention.blog/p/the-longest-onboarding-ever)) |
| **Loading screen** | "Создаём ваш персональный план..." (анимация 5–8 сек) | +10–20% конверсия (Noom pattern, [Retention Blog](https://www.retention.blog/p/the-longest-onboarding-ever)) |
| **Результат** | Профиль тяги ("Stress Eater, 3pm crash") + план + прогноз | Aha-moment до paywall ([Lenny's Newsletter](https://www.lennysnewsletter.com/p/mastering-onboarding-lauryn-isford)) |
| **Paywall** | Анимированный (2.9x), персонализированный (+17%), 2 тира, social proof, honest timeline, benefit CTA | Синтез данных из [PAYWALL-RESEARCH.md](./PAYWALL-RESEARCH.md) |
| **Push permission** | В контексте: "Включить предсказания тяги?" (после объяснения ценности) | Контекстный запрос > blind ask ([Appcues](https://www.appcues.com/blog/mobile-permission-priming)) |
| **Home** | Streak counter (видимый) + SOS кнопка + Today's lesson + Progress | Streaks 3.6x retention ([StriveCloud](https://www.strivecloud.io/blog/gamification-examples-boost-user-retention-duolingo)); SOS = core value |
| **SOS AI Chat** | Полноэкранный AI-разговор в момент тяги. Core "aha-moment" | Core value prop — Craving Defeated Rate >60% (Product Vision) |
| **Daily Check-in** | Как прошёл день? Сладкое: да/нет. Настроение. Быстро (<1 мин) | Daily engagement → 35% D30 ([Lucid](https://www.lucid.now/blog/retention-metrics-for-fitness-apps-industry-insights/)) |
| **Curriculum Lesson** | Урок дня (2–5 мин), neuroscience-backed | Сессии >5 мин → 35% D30 ([Lucid](https://www.lucid.now/blog/retention-metrics-for-fitness-apps-industry-insights/)) |
| **Progress** | Стрик, дни без сахара, достижения (milestones), тренды | Visual progress мотивирует ([Technology Rivers](https://technologyrivers.com/blog/what-every-healthtech-startup-can-learn-from-the-top-3-wellness-apps/)) |

## 6. Ключевые инсайты (топ-5)

1. **Paywall сразу после quiz, не откладывать.** 82–89% триалов = Day 0. Rootd: revenue x5 при раннем paywall. Greg: trial-to-paid с 3% до 15%. ([DEV Community](https://dev.to/paywallpro/the-paywall-timing-paradox-why-showing-your-price-upfront-can-5x-your-conversions-4alc))

2. **Quiz-based онбординг — это не "лишние шаги", а growth lever.** Noom ($750M+ ARR) имеет 80+ экранов. Каждый вопрос повышает perceived value и commitment. ([Retention Blog](https://www.retention.blog/p/the-longest-onboarding-ever))

3. **Streaks — самая ROI-эффективная retention-механика.** 3.6x long-term retention; Streak Freeze снижает churn на 21%. Внедрить первым делом в MVP. ([StriveCloud](https://www.strivecloud.io/blog/gamification-examples-boost-user-retention-duolingo))

4. **Тестирование pricing > тестирование визуала.** Pricing эксперименты дают 2–5x больший uplift. Приложения с A/B тестами зарабатывают в 40x больше. ([Adapty 2026](https://adapty.io/state-of-in-app-subscriptions/))

5. **ASO: "quit sugar app" = blue ocean.** Лидер Sugarfree имеет лишь 806 оценок (4.5). Sugar Quit с AI-дифференциацией, правильными keywords и 4.7+ рейтингом может доминировать в нише. ([App Store данные](https://apps.apple.com/us/app/sugarfree-quit-sugar-addiction/id1557550593))
