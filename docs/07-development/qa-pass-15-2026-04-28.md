### Free-tier paywall live — **partial**

Wiped Maya's `isPremium=true` to `false` + set `sosFreeLimit=3`, `sosUsedThisMonth=3` in AsyncStorage. App reloaded and the Supabase sync re-pulled Maya's profile from the server, restoring her premium status (server is the source of truth). Could not test free-tier paywall live without creating a fresh free user or editing Maya's row in Supabase. **Code-reviewed**: paywall logic uses `useUserStore.isPremium` for tier-aware UI; `shortPeak(peakHour)` for personalized peak hour copy. Both verified at code level + at the live render of onboarding paywall (step 14 above).
